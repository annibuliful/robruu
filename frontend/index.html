เรียนฟรีไม่เสียงเงิน
เพียงทำแบบฝึกให้บ่อย ก็ได้ Point ไปแลก เป็นบทเรียนได้แบบไม่ต้องเสียเงิน
หากต้องการเรียนต่อทันที
ก็สามารถเติมเงินเป็น Coin แล้วนำมาแลกเป็นบทเรียนได้ทันที
หากคุณเป็นผู้สอน
คุณก็สามารถสร้างบทเรียน
และหารายได้เสริมได้ง่าย




Point : คะแนนฟรี
เล่น Quiz ก็ได้ Point
ยิ่งคะแนนสูง Point ยิ่งสูงตาม







Coin : ซื้อง่าย จ่ายสะดวก เรียนได้ทันที
ซื้อ Coin ด้วยบัตร Truemoney
เติมเงินง่ายๆ ได้ด้วยบัตร Trumoney ที่หาซื้อได้ทั่วไป



แล้วนำ Coin ไปแลกเป็นบทเรียนได้ทันที



สำหรับผู้สอนที่ต้องการลงบทเรียน
ลงบทเรียนให้นักเรียนใช้งานฟรี
ร่วมพัฒนาการศึกษาให้แก่นักเรียน

หารายได้เสริมโดยการจัดขายคอร์ส
หากท่านมีบทเรียนที่ขายออนไลน์สำเร็จรูปอยู่แล้ว
สามารถนำมาลงเพิ่มกับเว็บรอบรู้ของเราได้
